# JollyAgent 演示

这个目录包含了 JollyAgent 的演示脚本和示例。

## 运行演示

在项目根目录下执行：

```bash
uv run python demo/demo.py
```

## 演示内容

演示脚本会展示以下功能：

1. **工具系统演示**
   - 显示所有可用的工具（run_shell, read_file, write_file）
   - 展示工具的参数信息

2. **基本对话演示**
   - 简单问候
   - 文件操作（创建demo.txt文件）
   - 读取文件内容
   - 执行系统命令（ls -la）

3. **ReAct循环演示**
   - 多步骤任务执行
   - 显示思考-行动-观察过程

4. **文件操作演示**
   - 创建多个文件
   - 列出目录内容
   - 批量读取文件

## 演示文件

演示过程中创建的所有文件都会保存在 `demo/` 目录中，不会污染项目根目录。

## 注意事项

- 确保已配置好硅基流动API的密钥
- 演示过程中会创建临时文件，可以随时删除
- 如果遇到错误，请检查日志输出
